# Decisive Meal Picker

## About

_Decision fatigue getting you down? Use Decisive Meal Picker to tell you where to lunch._

## Building the App

* Clone this repository
* Install required dependencies

  ```
  yarn install
  ```

* Run local server
  ```
  $ yarn start
  ```
